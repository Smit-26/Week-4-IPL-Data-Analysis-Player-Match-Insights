IPL Data Analysis 
This project analyzes IPL match and delivery datasets to extract key insights about player performance and team success using Python. The analysis identifies top run scorers calculates player strike rates computes team win percentages and studies player performance across seasons. The project also visualizes data using graphs to better understand trends and exports a summary file for further analysis. The datasets used are matches.csv containing match details like season teams and winner and deliveries.csv containing ball by ball match data with dataset paths /content/matches.csv and /content/deliveries.csv. The technologies used include Python Pandas Matplotlib and Seaborn. The output of the project includes visualizations of player and team performance along with an exported file player_summary.csv.
